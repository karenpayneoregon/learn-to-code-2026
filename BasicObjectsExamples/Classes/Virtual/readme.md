# About

The [virtual keyword](https://learn.microsoft.com/en-us/dotnet/csharp/language-reference/keywords/virtual) in C# is used to allow a method, property, indexer, or event in a base class to be overridden in a derived class, enabling polymorphism.

**Polymorphism**: The virtual keyword is a cornerstone of object-oriented programming (OOP) that enables polymorphism, allowing objects of a derived class to be treated as objects of a base class at runtime while still executing the derived class's specific implementation.
